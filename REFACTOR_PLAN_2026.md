# OPC 极客门户：微咨询降维打击重构蓝图

在“大厂经验 Skill 化 + OpenClaw 交付”的新统帅战略下，网站不再是展示“我做过什么”的画廊，而是一台精密运转的**“刺痛老板 -> 展示解决方案 -> 收割微咨询”**的销售漏斗机器。

以下是结合现有资产（两大沙盘、两大文案库）设计的全新转化路径。

## 📍 1. 核心链路与转化漏斗设计

整个网站的动线必须遵循一个极其冷酷的漏斗逻辑：
**提出极致痛点（Hero） -> 视觉震撼降维打击（沙盘 Demo） -> 证明深厚底蕴（项目库/日志） -> 收口变现（CTA）。**

### 模块 01：Hero 区（首屏痛点切入点）
- **废弃**：“我是前 58 大区总，寻找优质机会”。
- **重置为强烈 B 端痛点钩子**：全黑赛博风，极大的加粗标题，例如：
  > **“你团队 80% 的工作是在做无效的机械计算。”**
  > **“大厂不用明星销售打天下，大厂相信概率与机器。”**
- **动作**：下方紧跟一个极光流转的按钮：`[ 查阅大厂千万级管线流转图 ↓ ]`，引导视线下滑。

### 模块 02：大厂管线透视（沙盘全景展示）
这是转化信任的最佳武器。直接把这两个现有的强互动组件砸在老板脸上。
- **1号机位：【全链条利润模型沙盘】**
  - **定位**：展示“大厂是如何把线索像工业流水线一样榨干利润的”。
  - **展示法**：保留极客交互，让用户能放大拖拽看链路。
- **2号机位：【运营策略框架图】**
  - **定位**：展示“这是指挥大厂运转的中枢大脑逻辑”。
- **转化话术**：在这里抛出概念——“这套需要数千人维护的复杂沙盘，现在可以被精简封装为几十个 AI Skill，由机器代为执行。”

### 模块 03：武器库解密（项目经验文案库）
当老板被沙盘震撼后，他会质疑“这东西怎么落到我的小业务上？”此时祭出“项目库”。
- **表现形态**：将这个库更名为 **[微咨询落地方案库 / Skill Case Study]**。
- **层级逻辑**：
  - 外层是痛点卡片阵列：例如【线索流失治理】、【电销防飞单机制】。
  - 点击进入详情页，讲述**“如何将这一业务痛点，封装化、API 化，丢给 AI Agent 去自动化解决”。**
  - 核心：这里是业务壁垒展示区。

### 模块 04：数字极客底座（Vibe Coding 日记库）
- **表现形态**：作为网站的一个单独路由（如 `/build-in-public`），在首页底部区域做一个 Terminal / 弹幕形态的信息流走马灯。
- **功能定位**：
  1. 打造真实养成的“超级个体”极客 IP。
  2. 让目标受众相信你的代码与系统落地能力，托高咨询定价权。

### 模块 05：收口与留资（CTA)
- **重置**：“下载我的简历”。
- **变为**：“预约一次高维诊断会”。
  （例如“199元/半小时管线梳理微咨询（赠送某环节自动化脚本 Demo）”的入口）。

---

## 🛠️ 2. 执行切割路线表 (发配指令)

网站的改版必须以拆墙的形式，块状交付。**每次执行只 Focus 一个 Phase。**

**Phase 1: 首屏重写与门面切换 (预计耗时极短)**
- 修改 `page.tsx` 与 `Hero.tsx`。
- 将各种 `Hire me` / `Resume` 转变为 `System Build` / `Consultation`。
- 替换首屏文案。

**Phase 2: 沙盘组件就位 (排版重构)**
- 提取目前写好的两大 `Canvas` 或 `Flow` 组件，在页面的 C 位高亮和重组。
- 增加引导式的浮层文案。

**Phase 3: 路由与数据挂载 (重构详情页)**
- 调整路由架构，划分出 `/skills` (项目文案库) 和 `/logs` (Vibe 日志库)。
- 将底层数据或 Markdown 渲染组件映射对接好新的文案结构。
